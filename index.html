Projects

SwiftAid – Emergency Response System

React JS Express JS Node JS MongoDB

Developed SwiftAid, a web-based emergency response system for Sri Lanka. It handles SOS reporting, real-time responder assignment, and resource management for multiple user types (users, dispatchers, hospitals, NGOs).

The project was built using the MERN stack and followed the Agile Scrum methodology, emphasizing scalability, secure access, and integrations for live tracking and notifications.

QuickSpin – Laundry Management System

Java Java Servlets JSP MySQL Tailwind CSS

Created Quick Spin, a comprehensive web-based laundry management system. It allows customers to easily select services, book appointments, and process online payments.

Features a robust admin panel for managing services, verifying transactions, handling staff, and overseeing customer accounts, ensuring efficient operational control with a full Java/JSP/Servlet backend.

Ceylon Safari – Trip Management System

HTML CSS JavaScript PHP MySQL

Developed a web-based boat safari booking and management system, enabling users to view various packages, make online reservations, and manage their existing bookings.

The system was built using a standard **LAMP stack** setup (PHP and MySQL), featuring essential components like user authentication, a complete booking workflow, and administrator management tools.
